# Co-Parenting Portal

Bilingual (Chinese/English) co-parenting schedule and child support reference website.

**Live Site:** https://hanyunfan.github.io/co-parenting-portal

## Features

- **Custody Schedule** - ESPO (Extended Standard Possession Order) for Texas
- **RRISD Calendar** - Round Rock ISD school calendar with color-coded parent responsibility
- **Child Support Guide** - Texas Family Code citations for what's covered

## Tech Stack

- Pure HTML/CSS/JavaScript (no framework needed)
- Bilingual support (EN/CN)
- GitHub Pages hosting

## Legal Disclaimer

This website provides general information only and does not constitute legal advice. Consult a licensed Texas family law attorney for specific legal guidance.
